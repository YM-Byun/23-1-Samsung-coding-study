### 🍃 CONVENTION
- 언어는 `python`으로 통일합니다.
- 숙제 코드는 각자의 이름의 브랜치에 commit 합니다.
- 브랜치 이름은 각자의 이름으로 지정합니다.
- 코드 파일 이름은 `${num}-${name}` 포맷으로 권유합니다. (*파일 이름에 name은 꼭 포함할 것)
- 매주 화요일, 금요일 코드리뷰 후 `main`으로 merge 합니다. (*`Week${num}` 폴더에 commit할 것)
